How to run the movie trailer project. 

Note: I'm running my project on Windows 7 with Python 3.4.1

1. Download the project folder and place it in a directory you desire.
2. Open up your python Shell
3. In the file menu, click on the "open" option
4. Choose the entertainment_center.py and open it
5. Once opened, in the run menu, click on the "run" option.
6. Now, a web page will open with all my favorite movies.
